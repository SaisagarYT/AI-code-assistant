# AI CODEBASE ASSISTANT

> The AI codebase assistant enables developers to understand, explore and interact with software repositories using natural language. Insted of manually navigating through hundreds files and thousands of lines of code.<br>
Developers can ask about the codebase and recieve context-aware answers generated using Large Language Models (LLMs).
